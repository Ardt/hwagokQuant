# 퀀트 전략 레퍼런스

## 현재 구현: LSTM + 감성분석 + 매크로

### 아키텍처
```
입력 (60일 윈도우, 26개 피처)
  → LSTM (2층, 64 hidden, dropout 0.2)
  → FC (64 → 1)
  → Sigmoid
  → 이진 출력: P(내일 주가 상승)
```

### 왜 LSTM인가?
- 주가는 시계열 데이터 — 순서가 중요
- LSTM은 장기 의존성 포착 (예: 60일 추세 패턴)
- 가변 길이 패턴 처리 (횡보 → 돌파)
- 다중 피처 입력: 가격, 거래량, 지표, 매크로를 하나의 모델에서 결합

### 피처 그룹 (총 26개)

| 그룹 | 피처 | 목적 |
|------|------|------|
| 가격 (5) | 시가, 고가, 저가, 종가, 거래량 | 원시 시장 데이터 |
| 추세 (6) | SMA(20), SMA(50), EMA(12), MACD, MACD Signal, MACD Hist | 방향성과 모멘텀 |
| 모멘텀 (2) | RSI, Stochastic RSI | 과매수/과매도 |
| 변동성 (4) | BB High, BB Low, BB Width, ATR | 범위와 리스크 |
| 거래량 (1) | OBV | 매집/분산 |
| 수익률 (3) | 1일, 5일, 20일 변동성 | 최근 성과 |
| 감성 (1) | FinBERT 점수 (-1 ~ +1) | 뉴스 감성 (US만 적용) |
| 매크로 (4) | VIX, 금리, CPI, 실업률 | 경제 환경 |

### 학습 프로세스
```
1. Walk-forward 분할 (80% 학습 / 20% 검증, 시간순)
2. MinMaxScaler 정규화 (0-1)
3. 60일 슬라이딩 윈도우 → 시퀀스 생성
4. 이진 타겟: 내일 종가 > 오늘 종가
5. BCELoss + Adam 옵티마이저 (lr=0.001)
6. 조기 종료 (patience=10 에폭)
7. 검증 손실 기준 최적 모델 저장
```

### 시그널 생성
```
P(상승) > 0.5 → 매수
P(상승) < 0.5 → 매도 (즉, P(하락) > 0.5)
앙상블 조정 → 매수를 관망으로 하향 가능
```

### 앙상블 조정 (모델 후처리)
포트폴리오 상황에 따라 원시 시그널을 조정:

| 조건 | 효과 |
|------|------|
| VIX > 30 (공포 높음) | 매수 신뢰도 감소 (×0.6) |
| VIX > 40 (극단적) | 매수 신뢰도 감소 (×0.3) |
| 수익률 곡선 역전 (US) | 매수 신뢰도 감소 (×0.7) |
| 종목 집중도 > 25% | 매수 신뢰도 감소 (×0.5) |
| 현금 < 포트폴리오의 10% | 매수 신뢰도 감소 (×0.5) |
| 높은 공포 + 매도 시그널 | 더 적극적인 매도 |

조정된 확률이 0.5 미만으로 떨어지면 → 매수가 관망으로 변경.

### 백테스팅 규칙
- Walk-forward (미래 데이터 참조 없음)
- 손절: 진입가 대비 -5%
- 익절: 진입가 대비 +10%
- 포지션 사이징: 매수 시그널 간 균등 배분
- 지표: 총 수익률, 샤프 비율, 승률, 최대 낙폭

## 왜 이 전략인가?

### 강점
- **다중 시그널**: 기술적, 펀더멘털(매크로), 대안(감성) 데이터 결합
- **적응형**: LSTM이 시장 국면별 패턴 학습 (상승/하락/횡보)
- **리스크 인식**: 앙상블 레이어가 과집중 방지 및 매크로 환경 반영
- **종목별 모델**: 각 주식이 고유 모델을 가져 개별 특성 포착

### 약점
- **과적합 위험**: LSTM이 노이즈를 암기할 수 있음 (데이터 부족 시)
- **감성 공백 (KRX)**: 한국어 뉴스 감성분석 미구현
- **이진 단순화**: 실제 시장은 방향뿐 아니라 크기도 중요
- **지연**: 일봉 시그널만 가능, 장중 움직임 포착 불가
- **블랙박스**: 특정 시그널이 왜 생성됐는지 설명 어려움

### 완화 방안
- 조기 종료 + 드롭아웃 (0.2)으로 과적합 감소
- Walk-forward 검증 (랜덤 분할 절대 금지)
- 최근 데이터로 점진적 학습 (시장 국면 변화 적응)
- 앙상블 레이어가 블랙박스 위에 해석 가능한 규칙 추가
- 백테스트 지표로 실거래 전 신뢰도 확인

## 향후 전략 개선

### 단기
| 개선사항 | 영향 | 난이도 |
|----------|------|--------|
| 한국어 감성분석 (다국어 FinBERT) | KRX 정확도 +2-5% | 중 |
| 다중 클래스 출력 (상승/하락/횡보) | 거짓 시그널 감소 | 하 |
| 신뢰도 임계값 튜닝 | 정밀도/재현율 균형 개선 | 하 |
| 섹터 로테이션 시그널 | 상관 손실 회피 | 중 |

### 중기
| 개선사항 | 영향 | 난이도 |
|----------|------|--------|
| 어텐션 메커니즘 (Transformer) | 장거리 패턴 개선 | 중 |
| 교차 종목 피처 (섹터 ETF 수익률) | 종목별 시장 맥락 | 중 |

| 변동성 예측 (GARCH) | 포지션 사이징 개선 | 하 |

### 장기
| 개선사항 | 영향 | 난이도 |
|----------|------|--------|
| 그래프 신경망 (GNN) | 종목 간 관계 학습 | 상 |

| 다중 종목 LSTM | 단일 모델로 시장 역학 포착 | 상 |
| 옵션 오버레이 | 테일 리스크 헤지, 수익 창출 | 상 |

## 대안 전략 (미구현)

### 추세 추종
- 이동평균 교차 (골든크로스 / 데드크로스)
- 모멘텀 — 과거 수익률 기반 승자 매수, 패자 매도
- 돌파 — 지지/저항 돌파 시 매매

### 평균 회귀
- 페어 트레이딩 — 상관 자산의 괴리 시 롱/숏
- 볼린저 밴드 — 하단 매수, 상단 매도
- RSI 회귀 — 과매도 매수, 과매수 매도

### 통계적 차익거래
- 공적분 기반 페어 트레이딩
- 시장 중립 롱/숏 포트폴리오
- 횡단면 모멘텀/가치 팩터

### 팩터 투자
- Fama-French 팩터 (가치, 규모, 모멘텀, 수익성, 투자)
- 리스크 패리티
- 퀄리티 팩터 (ROE, 저부채, 안정적 이익)

### 고빈도 매매 (고급)
- 마켓 메이킹
- 주문 흐름 불균형
- 레이턴시 차익거래
